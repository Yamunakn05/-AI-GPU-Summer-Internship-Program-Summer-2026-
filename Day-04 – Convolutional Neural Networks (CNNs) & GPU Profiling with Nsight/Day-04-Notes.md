# Day 04 - Convolutional Neural Networks (CNNs) & GPU Profiling with Nsight

This session focused on Convolutional Neural Networks (CNNs), a deep learning architecture widely used for computer vision tasks. The session also introduced GPU profiling and performance analysis using NVIDIA Nsight tools.

* Image Classification
* Model Training and Evaluation
* GPU Profiling with NVIDIA Nsight
* Performance Optimization Techniques
