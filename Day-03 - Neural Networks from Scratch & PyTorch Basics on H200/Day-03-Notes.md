# Day 03 - Neural Networks from Scratch & PyTorch Basics on H200

## Overview

This session focused on understanding the fundamentals of neural networks and gaining hands-on experience with PyTorch for building and training deep learning models. The exercises were performed using GPU-accelerated computing resources.

* Introduction to Neural Networks
* Perceptrons and Artificial Neurons
* Forward Propagation
* Activation Functions
* Loss Functions
* Gradient Descent
* Introduction to PyTorch
* Tensor Operations in PyTorch
* GPU-Accelerated Deep Learning on H200

## Activities Completed

* Explored the architecture of neural networks.
* Worked with tensors and basic PyTorch operations.
* Implemented simple neural network components.
* Trained and evaluated basic deep learning models.
* Executed computations using GPU acceleration.

## Files Included

* Neural Network notebooks

* Hands-on exercises and assignments

## Key Learnings

* Understanding how neural networks learn from data.
* Role of activation functions and loss functions in model training.
* Working with tensors and datasets using PyTorch.
* Building and training simple deep learning models.
* Benefits of GPU acceleration for deep learning workloads.

Successfully gained foundational knowledge of neural networks and PyTorch, along with practical experience in training deep learning models using GPU resources.
